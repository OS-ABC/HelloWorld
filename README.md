# 欢迎来到OS-ABC/HelloWorld开源项目

**OS-ABC/HelloWorld是一个帮助大家学习开源技术的「入门级项目」**
任何一门IT技术或编程语言都是从HelloWorld开始学起，如果你想学习开源技术或通过开源项目学习软件开发技术，必定也要先从HelloWorld开始，以帮助你少走弯路。
